# 🎻 Julian Vance // Virtuoso Profile & Management Canvas 💼

A cinematic, responsive single-page web application featuring a dual-mode **Bento Grid layout**. The platform functions both as an immersive public-facing artist portfolio for a multi-instrumentalist virtuoso and a real-time data metrics dashboard for tour management logistics.

---

## 🚀 Core Features

* **Dual-State Engine:** Toggles seamlessly between a public **Artist Portfolio** mode (Gold aesthetic) and an enterprise **Manager Dashboard** mode (Indigo aesthetic) with instant data context shifting.
* **Responsive Bento Architecture:** Fully interactive grid canvas designed to adapt beautifully across mobile displays, tablets, and desktop setups.
* **Cinematic Hero Component:** Dynamic background interpolation and typography scaling that adapts based on the active operational view.
* **Logistics & Events Manifest:** High-performance data grids detailing live tour stops, ticket allocations, and real-time box office yields.

## 🛠️ Tech Stack & Architecture

* **Frontend Library:** React (Functional Components, Hooks API)
* **State Architecture:** Local component state tracking for smooth context swapping without page refreshes.
* **Styling Paradigm:** High-performance Inline JavaScript Styling coupled with custom CSS Flexbox wrappers to guarantee high performance and fast paint times.
* **Data Layer:** Decoupled data schemas (`gridData.js`) allowing rapid updates to text content, metrics, and event structures independently of component logic.
